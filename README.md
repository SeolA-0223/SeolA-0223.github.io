정적 GitHub Pages 사이트입니다.

## 배포용 저장소 이름

개인 GitHub Pages로 배포하려면 원격 저장소 이름은 아래 형식이어야 합니다.

`SeolA-0223.github.io`

현재 로컬 폴더 이름이 `SeolA-0223`여도 문제는 없습니다.

## 로컬 파일

- `index.html`: 메인 페이지
- `styles.css`: 스타일

## 배포 절차

1. GitHub에서 `SeolA-0223.github.io` 저장소를 만듭니다.
2. 이 폴더를 그 저장소에 연결합니다.
3. `main` 브랜치에 푸시합니다.
4. 사이트 주소는 `https://seola-0223.github.io/` 가 됩니다.
